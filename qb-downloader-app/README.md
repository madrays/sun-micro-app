# QB 下载器

qBittorrent WebUI 状态监控微应用，适用于 Sun-Panel。

## 功能

- 实时显示上传/下载速度
- 种子统计（下载中、做种、暂停、总数）
- 累计流量和分享率
- 磁盘剩余空间

## 支持尺寸

| 尺寸 | 像素 | 内容 |
|------|------|------|
| 1x1 | 76×76 | 上传/下载速度 |
| 1x2 | 174×76 | 速度 + 种子统计 |
| 2x1 | 76×174 | 速度卡片 + 统计 |
| 2x2 | 174×174 | 完整统计面板 |
| 2x4 | 370×174 | 全功能仪表盘 |

## 配置

在配置页面填写：
- qBittorrent WebUI 地址
- 用户名/密码
- 刷新间隔（默认 5 秒）
- 文字模式（亮色/暗色）

## 开发

```bash
npm install
npm run dev    # 开发
npm run pack   # 打包
```

## 权限

- `network`: 连接 qBittorrent WebUI
- `dataNode`: 存储配置
